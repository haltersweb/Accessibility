The Tests

  1. Print out the web page (for all breakpoints)
    • Make notes on the way flow should be
    • Make notes on semantic heading structure
    • Make note of all actionable elements (links and buttons)
  2. Run bookmarklet on page and run all tests in bottom blue bar that appears. This tests for:
    • reading order
    • heading hierarchy and validity
    • elements with click events that may need to be buttons instead
    • relative font sizing
    • form labeling and legending
